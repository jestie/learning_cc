🧠 What This Actually Is

The Learning AI — Command Centre is an interactive HTML dashboard designed to help AI enthusiasts and engineering teams run consistent, repeatable, and collaborative AI learning sessions.

Instead of:

random screen sharing,
45-minute rambling intros,
disappearing GitHub repos,
or “wait… which model were you using again?” moments…

…this gives the group:

a standard session structure,
architecture reference layers,
project templates,
shared learning principles,
and persistent session tracking.

Essentially:

“A command centre for organised AI nerding.”

✨ Features
⏱ Structured 30-Minute Session Agenda

A built-in learning session framework that keeps presentations focused and useful. Includes:

Quick framing
Live build walkthroughs
Key insights
Open discussion
Action commitments

No corporate waffle allowed.

🧱 AI Architecture Layer Mapping

Interactive expandable sections explaining the major layers of modern AI systems:

Foundation models
Prompt orchestration
Memory & retrieval
Agents & tools
Interfaces
Evaluation systems

Basically:

“The stack everyone pretends they fully understand on LinkedIn.”

📘 Built-In README Generator Template

Provides a standardised structure for documenting projects so future-you doesn’t hate present-you.

Because:

“I’ll remember how this works later”
is one of the greatest lies in software engineering.

📜 Group Operating Principles

Defines how the learning group functions:

Build consistently
Share openly
Ship rough ideas
Rotate presenters
Document everything
Keep compounding knowledge

Or in simpler terms:

“Touch grass occasionally, but mostly ship code.”

🗂 Persistent Session Log

Track sessions directly inside the interface using browser local storage:
